# Goals Module - User Guide

## Overview

The Tija Goals Module enables you to set, track, and evaluate performance goals aligned with organizational strategy. This guide covers how to use the Goals Module as an individual contributor.

## Getting Started

### Accessing the Goals Module

1. Navigate to **Goals** from the main menu
2. You'll see your Goals Dashboard with:
   - Active Goals
   - Completed Goals
   - Draft Goals
   - Performance Statistics

## Creating Goals

### Using the Goal Wizard

1. Click **"Create New Goal"** button on the dashboard
2. **Step 1: Select Template**
   - Browse suggested templates based on your job role
   - Or choose "Create Custom Goal" for a blank goal
3. **Step 2: Fill Goal Details**
   - Enter goal title
   - Select goal type (Strategic, OKR, or KPI)
   - Add description
   - Set start and end dates
   - Assign weight (percentage of total plan)
4. **Step 3: Review and Submit**
   - Review all details
   - Click "Create Goal"

### Goal Types

- **Strategic Goals**: Long-term, qualitative objectives (3-5 years)
- **OKRs (Objectives and Key Results)**: Short-term, agile goals with measurable key results
- **KPIs (Key Performance Indicators)**: Continuous metrics that track ongoing performance

## Managing Goals

### Viewing Goal Details

1. Click on any goal from your dashboard
2. View:
   - Goal information and description
   - Progress tracking
   - Alignment path (how it connects to organizational goals)
   - Evaluations and feedback
   - Performance score

### Updating Progress

1. Open the goal detail page
2. Update completion percentage
3. Update current values for KPIs
4. Add notes or comments
5. Save changes

### Goal Status

- **Draft**: Goal is being created, not yet active
- **Active**: Goal is in progress
- **Completed**: Goal has been achieved
- **On Hold**: Goal is temporarily paused
- **Cancelled**: Goal has been cancelled

## Submitting Evaluations

### Self-Evaluation

1. Navigate to a goal you own
2. Scroll to "Evaluations" section
3. Enter your score (0-100)
4. Add comments
5. Click "Submit Evaluation"

### Evaluating Others

If you're assigned as an evaluator (manager, peer, etc.):

1. You'll receive notifications for pending evaluations
2. Navigate to the goal detail page
3. Submit your evaluation with score and comments
4. Your evaluation will be included in the weighted score calculation

## Viewing Performance

### Dashboard Statistics

- **Active Goals**: Number of goals currently in progress
- **Completed**: Number of goals you've completed
- **Draft Goals**: Goals not yet activated
- **Average Completion**: Overall progress across all goals

### 360-Degree Feedback

View aggregated feedback from:
- Your Manager
- Yourself (Self-assessment)
- Peers
- Subordinates (for leadership goals)

### Performance Trends

View historical performance data to track improvement over time.

## Tips for Success

1. **Align with Strategy**: Use the alignment path to understand how your goals connect to organizational objectives
2. **Regular Updates**: Update progress regularly to maintain accurate tracking
3. **Use Templates**: Leverage goal templates for common objectives
4. **Set Realistic Weights**: Ensure goal weights sum to 100% across your plan
5. **Engage with Evaluators**: Provide context in comments to help evaluators understand your progress

## Frequently Asked Questions

**Q: Can I edit a goal after it's created?**
A: Yes, you can edit your own goals. Critical goals may require manager approval.

**Q: What happens if I don't meet a goal deadline?**
A: The system will track your progress. Discuss extensions or adjustments with your manager.

**Q: How are scores calculated?**
A: Scores use a weighted average from multiple evaluators (manager, peers, self) based on configured weights.

**Q: Can I see goals from other team members?**
A: You can only see goals you own or goals you're assigned to evaluate, based on permissions.
